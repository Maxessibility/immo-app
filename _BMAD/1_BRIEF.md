# 1. Brief Initial

## Contexte
Création d'une application web pour une agence immobilière indépendante. Le but est d'avoir une vitrine moderne pour les clients et un outil simple de gestion (back-office) pour l'agent immobilier.

## Objectifs Principaux
1.  **Vitrine Client :** Consulter les annonces immobilières (filtres, détails, photos).
2.  **Back-Office :** Interface sécurisée pour créer, modifier et supprimer des annonces.
3.  **Accessibilité :** Conformité stricte **WCAG 2.2 AA** (EAA compliance).
4.  **Modernité :** Design premium et fluide.

## Cibles
-   **Utilisateurs finaux :** Futurs acquéreurs ou locataires (navigation simple et accessible).
-   **Administrateur :** L'agent immobilier (gestion rapide des biens).

## Contraintes Techniques (Validées)
-   **Frontend :** React + Vite.
-   **Backend :** Node.js + Express.
-   **Base de données :** SQLite (pour le MVP/démarrage).
-   **Style :** Tailwind CSS.
-   **Langage :** TypeScript (Fullstack).

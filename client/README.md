# Immo App — Frontend

Frontend React (Vite + TypeScript) pour l'application immobilière.

## Scripts
- `npm run dev` : démarre le front en mode dev
- `npm run build` : build de production
- `npm run lint` : lint (inclut règles a11y)

## Notes
- Accessibilité WCAG 2.2 AA prise en compte dès le départ.
- Ce README remplace le template Vite par des infos projet utiles.
